## 📋 Sobre

Portfólio profissional desenvolvido para apresentar projetos, habilidades e experiência. Conta com design responsivo, alternância de tema claro/escuro e seções organizadas para facilitar a navegação.

### 🎯 Seções

- **👤 Sobre mim** — Breve apresentação pessoal e profissional
- **🛠️ Habilidades** — Tecnologias e ferramentas do dia a dia
- **📂 Projetos** — Galeria de projetos desenvolvidos
- **💼 Experiência** — Trajetória profissional e contribuições open source
- **📞 Contato** — Links para email, WhatsApp, LinkedIn e GitHub

### ✨ Features

- 🌓 **Tema escuro/claro** com persistência via `localStorage`
- 📱 **Design responsivo** para todos os dispositivos
- ⚡ **Hospedado na Vercel** com SPA rewrites

---

## 🚀 Deploy

O projeto está hospedado na Vercel:

🔗 **https://meu-portfolio-ebon-omega.vercel.app**

### Rodar localmente

```bash
# Clone o repositório
git clone https://github.com/ismaeldouglasdev/meu-portfolio.git

# Abra o index.html no navegador
open public/index.html
```
